When running on batteries, the device will go back to deep sleep after all events have been processed and the time is up
The device wakes up approximately every 3 hours to check if the time has passed to renew the IP address and deliver battery deliver statistics about the battery. Due to hardware/software restrictions, the time is not accurate ~±20%
To preserve battery life, set the value to 0 to disable auto wake up. If the device is used regularly, the statistics are updated once a day.
A single click is detected when the button is held less than Long Press Time. If the button is repeatedly pressed within Short Press Time in between, a double (or multi) click is detected. This value will increase the initial time before the first event is fired. If only single click is used, it should be set to 0
If a button is pressed longer then Long Press Time, a long press event is fired when releasing it before Repeat Time has been reached
If the button is held down for longer than Long Press Time + Repeat Time, a button held event is fired repeatedly in intervals of Repeat Time
